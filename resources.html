NetLogo Resources

NetLogo is a widely used, open source agent-based modeling language and environment. The software comes preloaded with a wide variety of models, including the models for this textbook. The models in NetLogo's models library are available online here. Additional NetLogo models submitted by members of the NetLogo community can be found on the NetLogo website (here) and on the Modeling Commons, an online community for NetLogo modelers. For additional information about NetLogo, please consult the NetLogo website and the online NetLogo User Manual.

NetLogo is developed and maintained by the Center for Connected Learning and Computer-based Modeling at Northwestern University. It is widely used in classrooms and research labs around the world. NetLogo curricula can be found here and an extensive list of research papers that use NetLogo is available here.

NetLogo Extensions

NetLogo was designed to be easily extendable. As a result there is a large number of extensions for NetLogo available that provide additional features to tool. NetLogo comes bundled with a number of extensions including a GIS extension, a sound extension, and a QuickTime for Java extension. Additional extensions are available on NetLogo's GitHub Extensions page. Notable non-bundled extensions include the Web extension that allows your to import/export data via HTTP, AH: are we not bundling NW?? the Networks extension that adds a number of network analysis primitives to the NetLogo language, and the custom logging extensions for tracking user interactions with NetLogo models.

Courses using NetLogo

There are thousands of courses all over the world that uses NetLogo. For a non-exhaustive list, please see the NetLogo website. You can also keep an eye out for NetLogo workshops near you on our Twitter.

NetLogo Tools

Since the initial release of NetLogo, members of the Center for Connected Learning and other researchers across the world have built specialized tools for NetLogo. Some of these help you design your own NetLogo-based classroom materials. Others help you build models, and analyze your data for research.

  • LevelSpace allows you to build models of models.
  • BehaviorSearch uses machine learning algorithms to help you scan the parameter space.
  • DeltaTick is a blocks-based interface for NetLogo.
  • NetTango takes NetLogo to touchscreens.
  • NetLogoLab allows you to connect physical devices to NetLogo.
